# global_superstore_analis_dashboard

# Цели проекта

- Построить полноценную BI‑систему для анализа продаж, клиентов, товаров и логистики.
- Выявить ключевые драйверы выручки и прибыли.
- Определить проблемные зоны: возвраты, низкомаржинальные товары, неэффективные скидки.
- Разработать собственную систему **выявления аномалий** на уровне заказов, клиентов, товаров и доставок.
- Создать презентабельный проект.

# Задачи проекта

**Подготовка данных**
- Очистка и нормализация данных.
- Создание корректной модели данных (Star Schema).
- Настройка связей между таблицами Orders, Products, Customers, Returns, Geography.

**Разработка визуальной аналитики**
- Создание 4 аналитических страниц: Overview, Orders & Customers, Product, Anomaly Detection.
- Построение динамических графиков, KPI‑карточек, матриц и точечных диаграмм.
- Настройка интерактивных фильтров и сегментов.

**DAX‑разработка**
- Создание мер для расчёта прибыли, маржи, среднего чека, частоты заказов.
- Создание флагов аномалий для заказов, клиентов, товаров и доставок.

**Аналитика и выводы**
- Анализ поведения клиентов и сегментов.
- Оценка прибыльности категорий и товаров.
- Анализ влияния скидок на прибыль.
- Выявление аномальных объектов и причин их появления.

## Anomaly & Outlier Detection — Выявление аномалий

**Anomaly & Outlier Detection**
 
![](images_&_video/Anomaly_&_Outlier_Detection.png)

https://github.com/user-attachments/assets/53b7457b-b891-4585-b729-fa515436b062

**Обнаружено:**
- **Аномальные заказы:** 0,23% (60 шт.)  
- **Аномальные клиенты:** 2,37% (413 чел.)  
- **Аномальные продукты:** 30,04% (1138 шт.)
- **Аномальные доставки:** 31,21% (8 тыс. шт.)

**Примеры аномалий:**
- Заказы с убытком до **–3791 ₽** при скидках 60–67%  
- Клиенты с высокой частотой заказов и отрицательной прибылью  
- Товары с маржой **–45%** и возвратами 30–40%  

**Выводы:**
- Основная причина аномалий — **чрезмерные скидки**  
- Аномальные товары концентрируются в **Office Supplies**  
- Аномальные доставки связаны с **долгим временем доставки**

# Общие аналитические выводы

- **Скидки** — ключевой фактор потери прибыли.  
- **Technology** — наиболее прибыльная категория.  
- **Office Supplies** — самая проблемная категория по аномалиям.  
- **Сегмент Consumer** — основной источник выручки.  
- **Аномалии в доставке** требуют оптимизации логистики.
