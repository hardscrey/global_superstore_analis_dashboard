# Dataset — Global Superstore

Данный проект построен на открытом датасете **Global Superstore**, содержащем информацию о продажах, клиентах, товарах, возвратах и географии.

---

# Состав данных

| Файл | Описание |
|------|----------|
| `Orders.csv` | Информация о заказах: даты, товары, скидки, прибыль, количество |
| `Customers.csv` | Данные о клиентах и сегментах |
| `Products.csv` | Категории, подкатегории и товары |
| `Returns.csv` | Информация о возвратах заказов |
| `Geography.csv` | Страны, города, регионы, рынки |

---

# Источник данных

Оригинальный датасет доступен на Kaggle:

**Global Superstore Dataset**  
https://www.kaggle.com/datasets/vivek468/global-super-store-dataset

---

# Как использовать данные

1. Скачайте файлы из папки `/data`.  
2. Откройте проект `global_superstore.pbix` в Power BI Desktop.  
3. При необходимости обновите пути к данным в Power Query:  
   **Home → Transform Data → Data Source Settings → Change Source**  
4. Обновите модель:  
   **Home → Refresh**
